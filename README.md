Project Insight – Stage-Based Mobile Application (Flutter)

This repository contains the Stage-1, Stage-2, and Stage-3 development of a cross-platform mobile application created using Flutter.
The final project demonstrates practical mobile development skills including UI design, device hardware integration, database connectivity, and accessibility features.

Overview

The goal of this project is to showcase progressive mobile-app development by building three stages:

✔ Stage 1 — Basic Flutter App

Focus on fundamental UI components and navigation.

✔ Stage 2 — Intermediate App

Includes login flow, navigation bar, themes, popups, tabs, scroll view & basic multimedia.

✔ Stage 3 — Final Smart App

Integrates advanced mobile features:

Camera

QR Scanner

Maps with Region of Interest (ROI)

Firebase database

Microphone speech input

Multi-language support

Permission handling

Magnifier mode

Screen reader / Text-to-Speech (TTS)

This GitHub repo contains the complete codebase, beginning from Stage 1 and expandable to Stage 3.
